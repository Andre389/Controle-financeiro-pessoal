# Controle Financeiro Pessoal - Deploy Vercel

Este pacote e uma versao estatica pronta para deploy na Vercel.

Arquivos:

- `index.html`: app completo em tema dark, conectado ao Supabase.
- `vercel.json`: configuracao simples para deploy estatico.

Como publicar no projeto antigo:

1. Abra o painel da Vercel no projeto `controle-financeiro-pessoal`.
2. Faca upload desta pasta ou do ZIP gerado.
3. Publique em Production.

Observacao: o app usa apenas a chave publica anon do Supabase. Nao inclui `service_role`.
Atualizado para deploy automático.
